Sincronização em Nuvem

Seus dados ficam salvos na nuvem, vinculados à conta com que você entrou. Faça login com o mesmo e-mail e senha em qualquer aparelho para acessar tudo. Cada conta só enxerga os próprios dados.

A nuvem ainda não está configurada. Abra o arquivo index.html, cole os dados do seu projeto no bloco window.FIREBASE_CONFIG e siga o passo a passo do arquivo GUIA-FIREBASE.md.
Na primeira vez, a conta é criada automaticamente. Use o mesmo e-mail e senha nos outros dispositivos.

Instalar como aplicativo

No computador, o sistema abre normalmente no navegador. No celular, você pode instalá-lo como um app nativo, com ícone na tela inicial e funcionamento em tela cheia.

No iPhone (Safari): toque em Compartilhar → "Adicionar à Tela de Início". No Android (Chrome): menu ⋮ → "Instalar app".

JSONBin — Geração de Licenças

O GestorPRO usa o JSONBin.io para gerar um Bin ID único como código de ativação de cada licença. Crie uma conta gratuita em jsonbin.io, acesse API Keys e cole sua Master Key abaixo.
